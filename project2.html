Project 2: Website Application

Introduction

Building on my previous skills from this portfolio website, I created a new website with a more practical application. Specifically, I designed a responsive website for Mountain Vista High School's Interact Club from scratch. This club prioritizes community service opportunities that I highlighted by including links to their sign ups. As the website coordinator for this club, I already have a preexisting website I made using Google Sites but this new website is built directly with HTML, CSS, and JavaScript. As a result, it may look more rudimentary as I had to construct it without the aid of Google Sites's more visual approach. Although it can be superior in some aspects, Google Sites did not allow me to add the Google Docs iframes like in my custom site making it easier for the user to see who is signed up for different events on the custom website. Description

It took me longer than expected to make the custom website because I had to learn how to add an image to the navigation bar without affecting other elements for each seperate page. The home page was also hard to format as I had to figure out how to add two images between the title and center some but not all of the elements. Nonetheless, it was a great learning experience and I am statisfied with the outcome. Although the custom website looks a little less complicated than the Google Sites website, I think this simple design makes it easier to navigate. I also enjoy how this custom website allows me to include Google Docs iframes on the Sign Ups page which was absent from the original website.

For this project, I specifically had to do a lot of extra research to figure out the formatting and ended up tweaking the seperate elements on the page about 20 times. I eventually had to scrap a section of the home page because even though it would look fine in my VSCode preview (the code editor), it would have wrapping problems when opened on a Google tab making it unreadable. My decision to abandon this section actually helped make the home page less cluttered so I am ultimately glad I made this decision.

Overall, I am satisfied with how my custom website turned out and feel it is an accurate demonstration of my acquired skills. In the future, I plan to continue to implement these skills by creating more custom websites possibly in the professional setting. From this point, I moved on to learn how to collect, manipulate, and store data into websites to eventually create built-in sign ups for my custom Interact Club website.
