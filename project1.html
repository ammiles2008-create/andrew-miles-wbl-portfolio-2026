Project 1: Learning Implementation

Introduction

Over the past few weeks, I have learned how to implement HTML, CSS, Javascript, and jQuery (a Javascript library that makes it easier to code in Javascript) to make this portfolio website. Below, I will share the learning processes I went through to make this Whac-A-Mole game. To play, press the start button and click the moles when they pop up to score points before time runs out. (You may have to resize your window to see the whole game).


Description

I first learned how to make the basic Hypertext Markup Language (HTML) elements for the website pages. HTML is effectively the "skeleton" for the website that codes for the elements of the website itself like the headers and paragraphs but doesn't style them. This makes most of the HTML elements look bland but overall create the core functionality of the website. For example, the title of this webpage, the navigation bar, the Whac-A-Mole game’s shapes, and the text you are reading right now were all created by me using HTML.

To make the text and the graphics look visually appealing, I had to apply a Cascading Style Sheets (CSS) layer on top of the HTML. I could have theoretically done all of the styling within the HTML, but that would have been too tedious and it wouldn’t have let me apply styles to all of the same elements on the page so I opted to use CSS instead. CSS only focuses on styling the HTML so it can style all of the HTML elements on a page, some HTML elements with a specific tag or class, or any combination of those methods. I used this CSS to create the light blue background for this page, the curvature and background colors for the buttons and the Whac-A-Mole game, and the different font colors for the titles and my body text (this text). I was also able to use CSS to make navigation bar interactions and the title animations so the user can switch between the pages and to stylize the pages further with growing titles.

Although useful, the CSS interactions and animations are harder to code and less customizable than their Javascript counterparts so I also learned how to use Javascript to make my pages more responsive. I already have a strong background knowledge of Java from my AP Computer Science A course last year so Javascript was relatively easy to learn as it uses the same logic structure although the specific commands to access the HTML and CSS were hard to memorize and implement in this coding language. As a solution, I learned how to use JQuery, a Java library, which helped me simplify the coding commands for Javascript without sacrificing the logic system I already understood. I was then able to use this combination of Javascript and JQuery to make the Whac-A-Mole game function properly and develop the complex animations for this game.

In all, I learned how to use HTML, CSS, Javascript, and JQuery to build this originally bland white background into a stylized, interactive, and fun website accessible from anywhere (with the Internet).
